# Astro Site App

This workspace contains the static Astro app for the Sheridan Family Cookbook.

The app renders public recipe pages, family source pages, search data, and a public email-based recipe submission page. There are no production login, upload, or admin routes.

## Important Paths

```text
apps/site/
├── astro.config.mjs
├── package.json
├── public/
│   └── resources/        # Generated by npm run sync
└── src/
    ├── content/
    │   └── recipes/      # Generated by npm run sync
    ├── layouts/
    ├── lib/
    │   ├── nav.ts
    │   ├── site-config.ts
    │   └── sources.ts
    └── pages/
```

Generated folders are ignored by git:

- `src/content/recipes/`
- `public/resources/`
- `dist/`
- `.astro/`

Edit source recipes in `../../content/recipes/`, not in `src/content/recipes/`.

## Rendering Model

Astro builds a fully static site into `apps/site/dist/`.

Public routes:

- `/`
- `/recipes/**`
- `/about/`
- `/family-sources/`
- `/add-recipe/`
- `/search.json`
- `/resources/**`

## Site Config

Edit `src/lib/site-config.ts` for site-level values:

```ts
export const siteConfig = {
  name: 'Sheridan Family Cookbook',
  url: 'https://cookbook.cyberhelm.com',
  submissionEmail: 'cookbook@cyberhelm.com',
};
```

The `/add-recipe/` page uses `submissionEmail` to build the `mailto:` link.

## App Commands

Most commands should be run from the repo root:

```sh
npm run sync
npm run dev
npm run build
npm test
npm run preview
```

Direct workspace commands are also available:

```sh
npm run dev -w sheridan-cookbook-site
npm run build -w sheridan-cookbook-site
npm run preview -w sheridan-cookbook-site
```
